•  اذهب إلى Settings → Pages
•  في قسم Source اختر Deploy from a branch
•  اختر main branch و /(root)
•  اضغط Save
5.  احصل على الرابط
بعد 2-3 دقائق، سيظهر لك رابط مثل:
